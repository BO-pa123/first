Traditionally, fragment identifier links are those clickable items in your page that do not require the browser to request anything new from the server. These links allow users to jump to certain targeted sections of your web page.

(#section3) Section 3

There are two steps to creating such links: marking something as a possible target of a link and creating the link to that target.

(#section4) Section 4

The modern HTML way of marking content as a possible target of a link is to assign an id attribute to the element that wraps that content. As you know by now, the value of an id attribute of any element within a single HTML page must be unique. Therefore, assigning an id attribute to an element with a unique value creates a unique target name as well.
